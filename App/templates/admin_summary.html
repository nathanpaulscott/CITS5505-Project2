Quiz List
Instructions
Click on a Quiz row to edit it.
Use the import button to import one or more quizes and/or images.
Use the export button to export one or more quizes.
Use the delete button to enter a list of quiz ids for deletion.
NOTE: export a quiz to get the format required for import.
NOTE: quiz should use .quiz extension, images can use any extension.
NOTE: if you import a quiz with and existing name, it will overwrite that quiz in the DB.
NOTE: you can see in the missing images column the number of missing images for a quiz, edit the quiz to see which images.
NOTE: when selecting files for import, non .quiz files are assumed to be images and are deleted if not required by any quiz.

Give the comma delimited list of Quiz Ids to Export or 'all' to export all Quizes

  

Give the comma delimited list of Quiz Ids to Delete them from the Database
